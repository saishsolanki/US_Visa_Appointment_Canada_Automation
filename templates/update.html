How it works: Clicking Pull & Update runs git pull origin main on the server, then installs any new Python dependencies from requirements.txt. After the update finishes, restart the visa checker for the changes to take effect. Your config.ini is never touched.

Pull Latest Code

This will fetch the latest changes from the main branch of your GitHub repository and update all Python dependencies.

Make sure the server has network access to GitHub and that the repository was cloned with write permissions (or at least pull access).
